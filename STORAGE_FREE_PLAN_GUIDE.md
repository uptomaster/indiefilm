# Firebase Storage 무료 플랜 사용 가이드

## 💡 무료 플랜에서 Storage 사용하기

Firebase Storage는 **무료 플랜(Spark)에서도 사용 가능**합니다! 하지만 초기 활성화 시 Blaze 플랜으로 업그레이드하라는 메시지가 나올 수 있습니다.

### ✅ 해결 방법

#### 방법 1: Storage 활성화 (권장)

1. Firebase 콘솔 접속
2. 프로젝트 설정 → 사용량 및 청구
3. **Blaze 플랜으로 업그레이드** 클릭
4. 결제 정보 입력 (신용카드 등록 필요)
5. **무료 할당량 내에서는 요금이 부과되지 않습니다!**

**무료 할당량:**
- 저장: 5GB/일
- 다운로드: 1GB/일
- 업로드: 20GB/일

→ 일반적인 사용으로는 충분합니다!

#### 방법 2: Storage 없이 사용 (현재 방식)

Storage를 활성화하지 않아도 **URL 입력 방식**으로 이미지를 사용할 수 있습니다:

1. 프로필 사진 URL 입력 필드 사용
2. 외부 이미지 호스팅 서비스 활용:
   - Imgur (https://imgur.com)
   - Cloudinary (https://cloudinary.com)
   - GitHub (이미지 파일을 저장소에 업로드)
   - Google Drive (공유 링크)

### 🔄 현재 구현 상태

현재 앱은 **두 가지 방식 모두 지원**합니다:

1. **파일 업로드** (Storage 활성화 시)
   - Firebase Storage에 직접 업로드
   - 자동으로 URL 생성

2. **URL 입력** (Storage 없이도 가능)
   - 외부 이미지 URL 입력
   - 즉시 사용 가능

### 💰 비용 걱정 없이 사용하기

- **무료 할당량 내에서는 요금이 부과되지 않습니다**
- 사용량이 적으면 계속 무료로 사용 가능
- 사용량 초과 시에만 요금 발생 (사전 알림 제공)

### 🎯 권장사항

개인 프로젝트나 포트폴리오라면:
- **Blaze 플랜으로 업그레이드** (결제 정보만 등록, 실제 요금은 무료 할당량 내에서 발생하지 않음)
- 또는 **URL 입력 방식**으로 계속 사용

상업적 서비스라면:
- 사용량 모니터링
- 무료 할당량 내에서 사용
- 필요시 유료 플랜 고려
