# 배우 기능 완료 ✅

## 완료된 기능

### 1. 배우 목록 페이지 (`/actors`)
- **필름/영화사 느낌의 다크 테마 디자인**
- 히어로 섹션: "CASTING" 타이틀 + 골드 악센트
- 검색 기능: 이름, 스킬, 경력으로 검색
- 필터링: 지역, 나이대별 필터
- 배우 카드: 
  - 큰 프로필 사진 (없으면 기본 아이콘)
  - 호버 시 그라데이션 오버레이 효과
  - 시네마틱한 그림자 효과
  - 골드 악센트 색상

### 2. 배우 상세 페이지 (`/actors/[id]`)
- **시네마틱한 히어로 섹션**
  - 전체 화면 프로필 사진
  - 필름 그라데이션 오버레이
  - 하단 정보 오버레이 (골드 텍스트)
- 데모 릴 임베드 (YouTube/Vimeo)
- 경력, 스킬, 자기소개 섹션
- 갤러리 (사진 여러 장)
- 캐스팅 제안 버튼 (제작자 전용)

### 3. 배우 프로필 생성/수정 폼 (`/actors/me`)
- **필름 느낌의 다크 폼 디자인**
- 기본 정보: 예명, 나이대, 키, 체형, 지역
- 자기소개 (텍스트 영역)
- 프로필 사진 URL 입력 (Storage 없이도 동작)
- 데모 릴 URL 입력 (YouTube/Vimeo)
- 연기 경력 (동적 배열 추가/삭제)
- 특기/스킬 (동적 배열 추가/삭제)
- 공개 설정 (isPublic 체크박스)

## 디자인 컨셉

### 색상 팔레트
- **베이스**: 블랙 (#000000), 다크 그레이 (#1a1a1a)
- **악센트**: 골드 (#D4AF37), 옐로우 (#F5C842)
- **텍스트**: 화이트, 그레이 (#9ca3af)

### 주요 디자인 요소
- 필름 스트립 패턴 배경
- 그라데이션 오버레이 (film-gradient, film-overlay)
- 시네마틱한 그림자 (cinematic-shadow)
- 골드 테두리 및 악센트
- 호버 시 스케일/트랜스폼 효과

### 커스텀 CSS 클래스
- `.film-gradient`: 그라데이션 오버레이
- `.film-overlay`: 하단 그라데이션
- `.film-gold`: 골드 텍스트 색상
- `.film-strip`: 필름 스트립 패턴
- `.cinematic-shadow`: 드라마틱한 그림자
- `.actor-card`: 배우 카드 호버 효과
