# IndieFilm Hub 개선 계획

## 🎯 현재 상태 분석

### ✅ 완료된 핵심 기능
1. **인증 시스템**: 로그인/회원가입/역할 선택
2. **배우 프로필**: 생성/수정/조회/검색/필터링
3. **영화 관리**: 업로드/수정/목록/상세/평점
4. **제작자 프로필**: 개인/팀 프로필
5. **요청 시스템**: 캐스팅 제안/출연 희망
6. **실시간 채팅**: 요청 관련 채팅
7. **커뮤니티**: 게시판/댓글
8. **영화 상태**: 제작중/제작예정/제작완료

---

## 🚀 우선순위별 개선 사항

### 🔴 우선순위 1: 핵심 UX 개선 (즉시 적용)

#### 1.1 검색 기능 강화
- [ ] **통합 검색바** 추가 (헤더/사이드바)
  - 영화 제목, 배우 이름, 제작자 이름 통합 검색
  - 실시간 검색 제안 (autocomplete)
  - 검색 결과 페이지 (`/search?q=...`)
- [ ] **고급 검색 필터**
  - 영화: 장르, 연도, 상태, 평점 범위
  - 배우: 지역, 나이대, 키, 스킬
  - 제작자: 타입(개인/팀), 지역, 전문 분야

#### 1.2 알림 시스템
- [ ] **실시간 알림** (Firebase Cloud Messaging 또는 Firestore 리스너)
  - 새로운 요청 알림
  - 요청 수락/거절 알림
  - 댓글 알림
  - 채팅 메시지 알림
- [ ] **알림 센터** (`/notifications`)
  - 읽음/안읽음 표시
  - 알림 타입별 필터링
  - 일괄 읽음 처리

#### 1.3 에러 처리 개선
- [ ] **전역 에러 바운더리** 추가
- [ ] **사용자 친화적 에러 메시지**
  - 기술적 에러를 일반 사용자가 이해할 수 있는 메시지로 변환
  - 재시도 버튼 제공
- [ ] **에러 로깅** (Sentry 또는 Firebase Crashlytics)

#### 1.4 로딩 상태 개선
- [ ] **스켈레톤 UI** 추가 (로딩 중 콘텐츠 구조 미리보기)
- [ ] **로딩 진행률 표시** (이미지 업로드 등)
- [ ] **최적화된 로딩 순서**
  - 중요 콘텐츠 우선 로드
  - 지연 로딩 (lazy loading)

---

### 🟡 우선순위 2: 기능 강화 (단기)

#### 2.1 정렬 및 필터링 개선
- [ ] **다중 정렬 옵션**
  - 영화: 최신순/인기순/평점순/제목순
  - 배우: 최신순/이름순/인기순
  - 커뮤니티: 최신순/인기순/댓글순
- [ ] **저장된 필터** (사용자별 선호 필터 저장)
- [ ] **URL 쿼리 파라미터**로 필터 상태 공유

#### 2.2 이미지 최적화
- [ ] **이미지 리사이징** (업로드 시 자동 리사이즈)
- [ ] **WebP 형식 지원**
- [ ] **Lazy loading** (Intersection Observer)
- [ ] **Blur placeholder** (이미지 로딩 전)

#### 2.3 성능 최적화
- [ ] **React.memo** 적용 (불필요한 리렌더링 방지)
- [ ] **가상화 (Virtualization)** (긴 목록)
- [ ] **코드 스플리팅** (페이지별)
- [ ] **이미지 CDN** (Cloudinary 또는 Firebase Hosting)

#### 2.4 데이터 검증 강화
- [ ] **클라이언트 사이드 검증** 강화
- [ ] **서버 사이드 검증** (Firebase Functions)
- [ ] **입력 필드 실시간 검증**
- [ ] **파일 크기/형식 검증**
